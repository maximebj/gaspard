Le Bon Français convertit automatiquement les apostrophes, guillemets, points de suspensions en leurs équivalents Français. Il ajoute également des espaces insécables avant les doubles ponctuations. À votre service !

== Description ==

Cette extension pour le nouvel éditeur WordPress 5.0, Gutenberg, est indispensable à tous les sites Français. 

Elle permet de convertir automatiquement les apostrophes, guillemets, points de suspension lors de la saisie en leurs équivalents Français.

Elle également des espaces insécables avant les doubles ponctuations afin d’éviter que des point d’exclamation ou d’interrogation ne reviennent à la ligne seuls.

=== Multilangue ===

Compatible multilangue (WPML, Polylang) : l’extension se désactive pour les contenus non Français.

=== Exclusion de blocs ===

Vous pouvez également exclure des blocs de la conversion afin d’éviter des soucis, comme avec les blocs code ou html par exemple.
